CleanArchitecture.Java Web UI built with:

  • Eclipse STS 4.5 and Java 8 for server-side code
  • Angular 8.1.3 and TypeScript for client-side code
  • Bootstrap for layout and styling

 

This Eclipse project consists of an Angular UI built on Node 10.15.3. The UI serves data from an MSSQL 2017 database via a Java REST API developed with SOLID design & Clean Architecture principles. The data consists of the following:

  • Categories: Category table which contains product categorizations.
  • Products: Product table which contains product definitions.

The UI facilitates CRUD operations to the database tables through the API.
